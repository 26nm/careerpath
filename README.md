# CareerPath: A Job Search Management Tool

CareerPath is a full-stack web application designed to streamline the job search process by combining application tracking, interview scheduling, and AI-powered resume analysis into one unified platform.  

🔗 https://careerpath-cad61.web.app/

---

## 🚀 Features

### Resume Uploading & Parsing
- Upload resumes (.pdf, .doc, .docx, .txt)  
- Automatically parsed using backend processing  
- Stored securely in Firebase Storage and Firestore   

### AI-Powered Skill Extraction
- Uses AI to extract relevant sills from resumes and job descriptions  
- Works across multiple domains (tech, retail, hospitality, etc.)  
- Eliminates need for manual keyword filtering

### Resume Analysis
- Compares extracted resume skills against job descriptions  
- Displays match rate and matching/missing skills  
- Designed for clarity and actionable insights

### Interview Scheduler
- Track upcoming interviews  
- Automatic status updates (upcoming, completed, etc.)  
- Organized and time-aware interface  

### Job Application Tracker 
- Manage job applications in one place  
- Track statuses and progress  
- Clean and structured UI
